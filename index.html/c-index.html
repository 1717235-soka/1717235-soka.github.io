総評

最近の梅おにぎりは、梅が酸っぱすぎず甘すぎずマイルドでおいしいです。個人的には梅＋αで具が入っているのが好きですが、そういったおにぎりは食べづらいのが難点です。

　

おにぎりの成分表収集をしていると、おにぎりの味より、ラベルがうまく剥がせた時のほうの喜びが大きくなってしまいました。
